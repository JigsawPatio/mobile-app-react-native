# mobile-app-react-native

## Description

This is a mobile application built using React Native, a framework for building native mobile apps using JavaScript and React. This project showcases a fully functional mobile app that demonstrates various features and functionalities.

## Features

*   **Home Screen**: Displays a list of items
*   **Item Details**: Shows detailed information about each item
*   **Search Bar**: Allows users to search for items
*   **Item Sorting**: Enables users to sort items by name or price
*   **Item Filtering**: Enables users to filter items by category

## Technologies Used

*   **React Native**: A framework for building native mobile apps using JavaScript and React
*   **JavaScript**: Used for writing application logic and UI components
*   **ES6/ES7**: Used for writing modern JavaScript code
*   **Node.js**: Used for building and running the application
*   **npm**: Used for managing dependencies
*   **Xcode**: Used for building and debugging the iOS version of the app
*   **Android Studio**: Used for building and debugging the Android version of the app

## Installation

### Prerequisites

*   Node.js (14 or higher)
*   npm (6 or higher)
*   Xcode (for iOS)
*   Android Studio (for Android)

### Installation Steps

1.  Clone the repository using the following command:

    ```bash
git clone https://github.com/your-username/mobile-app-react-native.git
```

2.  Navigate to the project directory using the following command:

    ```bash
cd mobile-app-react-native
```

3.  Install the dependencies using the following command:

    ```bash
npm install
```

4.  Start the development server using the following command:

    ```bash
npm start
```

5.  Install the required platforms and packages using the following command:

    ```bash
npx react-native run-ios
```

    or

    ```bash
npx react-native run-android
```

### Troubleshooting

*   **iOS:**

    *   Make sure you have the latest version of Xcode installed.
    *   Ensure that the simulator is selected in the Product > Destination menu.
*   **Android:**

    *   Make sure you have the latest version of Android Studio installed.
    *   Ensure that the device or emulator is selected in the Run > Manage Configurations menu.
